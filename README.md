# ELEC3612 Assignment 1 — Body Signal of Smoking

## Dataset
Body Signal of Smoking — https://www.kaggle.com/datasets/kukuroo3/body-signal-of-smoking

## Methodology
1. Data loading via KaggleHub API
2. Preprocessing: label encoding, StandardScaler normalisation, duplicate removal
3. Supervised models: Decision Tree, Random Forest, Logistic Regression, SVM
4. Unsupervised: K-Means Clustering (k=2)
5. Ensemble methods: Bagging, AdaBoost, Gradient Boosting, Voting, Stacking

## How to Run
1. Open `ELEC3612_Assignment1.ipynb` in Google Colab
2. Run all cells in order (Runtime → Run all)
